# CIFAR-10 Image Classification using ANN & CNN

## 📌 Project Overview
This project demonstrates image classification on the CIFAR-10 dataset using two Deep Learning models:

- Artificial Neural Network (ANN)
- Convolutional Neural Network (CNN)

The project compares the performance of both models and shows why CNN performs better for image classification tasks.

---

## 📂 Dataset

**CIFAR-10** is a popular image classification dataset containing:

- 60,000 color images
- 10 image classes
- 50,000 training images
- 10,000 testing images
- Image size: 32 × 32 pixels

Classes:
- Airplane
- Automobile
- Bird
- Cat
- Deer
- Dog
- Frog
- Horse
- Ship
- Truck

---

## 🚀 Project Features

- Load and preprocess the CIFAR-10 dataset
- Build an ANN model
- Build a CNN model
- Compare ANN and CNN performance
- Plot validation accuracy
- Apply Data Augmentation
- Generate a final comparison table
